## Description

*(4,000-character limit)*

Apricot is a Bitcoin address and transaction explorer. Look up any public Bitcoin address to see its balance, transaction history, and activity summary. Tap into any transaction to understand where funds moved, how much was paid in fees, and how many confirmations it has received.

**What Apricot shows:**
- Address balance, total received, and total sent
- Full transaction list for any public address
- Transaction inputs and outputs with amounts
- Confirmation count and block height
- Network fees

**Designed to be clear, not confusing.**
Blockchain data is dense. Apricot presents it in plain language with a calm, readable interface — no charts, no price tickers, no noise.

**Public data only.**
Apricot reads publicly available information from the Bitcoin blockchain via the mempool.space API. It does not connect to any wallet, does not store private keys, and does not have access to any funds.

**Disclaimer:**
Apricot is an educational tool for exploring public Bitcoin blockchain data. It does not provide financial advice, investment recommendations, price information, or trading functionality. Nothing in Apricot should be interpreted as a recommendation to buy, sell, or hold Bitcoin or any other asset.
