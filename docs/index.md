# Apricot

**Apricot** is a visual Bitcoin address and transaction explorer for iOS.

Look up any public Bitcoin address to see its balance, transaction history, and activity summary. Tap into any transaction to understand where funds moved, how fees were paid, and how many confirmations it has received.

---

## What Apricot Does

- Search any public Bitcoin address
- View balance, total received, and total sent
- Browse the full transaction list for an address
- Open any transaction to see inputs, outputs, fees, and confirmations
- All data is publicly available on-chain — no wallet connection required

## What Apricot Does Not Do

- No wallet connection
- No private keys
- No account or sign-up
- No trading or financial advice
- No price information

---

Apricot is an educational tool for exploring public Bitcoin blockchain data.
Data is sourced from the [mempool.space](https://mempool.space) public API.
