# Support — Apricot

**Apricot** is a visual Bitcoin address and transaction explorer for iOS. Look up any public Bitcoin address or transaction — no wallet connection, no sign-up.

---

## Reporting an Issue

If you encounter a bug, unexpected behavior, or have a question, please reach out:

**Email:** [apricotappsupport@gmail.com](mailto:apricotappsupport@gmail.com)

When reporting an issue, it helps to include:
- A description of what you expected to happen
- A description of what actually happened
- The iOS version and device model (e.g., iPhone 15, iOS 17.4)
- The Apricot app version (visible in Settings → General → iPhone Storage → Apricot)

---

## Frequently Asked Questions

**Does Apricot connect to my wallet?**
No. Apricot reads publicly available Bitcoin blockchain data only. It never connects to a wallet, requests private keys, or has access to any funds.

**Where does the data come from?**
Apricot uses the [mempool.space](https://mempool.space) public API to fetch on-chain data.

**Why is a transaction or address missing or incorrect?**
Blockchain data can take time to propagate. If a very recent transaction is not showing, try again in a few minutes. For persistent issues, contact us.

**How do I clear my recent searches?**
Recent searches are stored locally on your device. You can clear them by deleting and reinstalling the app.
